# Demographic Data Analyzer (Analisador de Dados Demográficos)


## Sobre
Este é um projeto desenvolvido como parte do curso "Data Analysis with Python" da [freeCodeCamp](www.freecodecamp.org).

O objetivo é usar a biblioteca Pandas para analisar um conjunto de dados do Censo dos EUA (`adult.csv`) 
e responder a várias perguntas sobre a demografia e renda da população estudada.


## Estrutura

O projeto calcula as seguintes métricas:

1.  Número de pessoas por raça.
2.  Idade média dos homens.
3.  Percentual de pessoas com diploma de Bacharelado (`Bachelors`).
4.  Percentual de pessoas com educação avançada (`Bachelors`, `Masters`, ou `Doctorate`) que ganham mais de 50 mil anuais.
5.  Percentual de pessoas sem educação avançada que ganham mais de 50 mil anuais.
6.  Número mínimo de horas que uma pessoa trabalha por semana.
7.  Percentual de pessoas que trabalham o mínimo de horas por semana e ganham mais de 50 mil anuais.
8.  O país com a maior porcentagem de pessoas ganhando mais de 50 mil.
9.  A ocupação mais popular para aqueles que ganham mais de 50 mil na Índia.
  

## Instrução

1.  Clone este repositório (e certifique-se de ter o arquivo `adult.csv` no mesmo diretório):
    ```bash
    git clone github.com
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd demographic-data-analyzer
    ```
3.  Instale a biblioteca pandas, se ainda não tiver:
    ```bash
    pip install pandas
    ```
4.  Execute o arquivo `main.py` para rodar a análise:
    ```bash
    python3 main.py
    ```

## Arquivos no Projeto

*   `demographic_data_analyzer.py`: Contém a função principal `calculate_demographic_data()`.
*   `main.py`: Arquivo de exemplo para testar a função.
*   `adult.csv`: O conjunto de dados utilizado na análise.
